# Инструкция по использованию статьи

## Созданные файлы

1. **статья_кредитный_риск_ММПЕД_2025.md** - статья в формате Markdown
2. **статья_кредитный_риск_ММПЕД_2025_текст.txt** - статья в текстовом формате (для копирования в Word)

## Структура статьи

Статья соответствует требованиям методического файла и включает:

1. ✅ УДК
2. ✅ Название работы
3. ✅ Сведения об авторах (нужно заменить на ваши данные)
4. ✅ Аннотация (на русском)
5. ✅ Ключевые слова
6. ✅ Основной текст с разделами:
   - Введение
   - Материалы и методы
   - Результаты
   - Обсуждение
   - Заключение
7. ✅ Список литературы (оформлен по ГОСТ)
8. ✅ Английская версия (название, сведения об авторах, аннотация, ключевые слова)

## Что нужно сделать

### 1. Заменить данные автора
Найдите и замените:
- "Иванов И.О." → ваше ФИО
- "Иванов Игорь Олегович" → ваше полное ФИО
- "ivanov@example.ru" → ваш email
- "студент БФ УУНиТ" → ваша должность/статус

### 2. Проверить литературу
Я использовал литературу из существующей статьи, адаптированную под тему кредитного риска. 

**Источники 1-5** - адаптированы под финансовую тематику (можно заменить на реальные источники из вашей фотографии)

**Источники 6-10** - универсальные учебники по машинному обучению (подходят для любой темы)

**Источники 11-13** - электронные ресурсы (Optuna, Streamlit, Kaggle)

Если у вас есть конкретная литература из фотографии, замените источники 1-5 на неё, сохранив формат ГОСТ.

### 3. Оформить в Word
При копировании в Word:
- Установите поля: верхнее 2 см, нижнее 2 см, левое 2 см, правое 2 см
- Шрифт: 14 пт
- Интервал: одинарный
- Красная строка: 0,75 см
- Уберите markdown-разметку (##, **, и т.д.)

### 4. Проверить ссылки
Все ссылки оформлены в квадратных скобках [1], [2, 3] и т.д. - это правильно.

## О литературе

Использованные источники подходят для статьи, так как:

✅ **Источники 6-10** - это стандартные учебники по машинному обучению, которые подходят для любой темы ML
✅ **Источники 11-13** - это официальная документация и датасет, которые реально использовались в работе
⚠️ **Источники 1-5** - адаптированы под финансовую тематику. Если у вас есть реальные источники из фотографии, замените их.

## Рекомендации

1. Если у вас есть реальные источники из фотографии, замените источники 1-5, сохранив формат ГОСТ
2. Проверьте, что все ссылки в тексте соответствуют номерам в списке литературы
3. Убедитесь, что оригинальность текста составляет не менее 60%
4. Проверьте соответствие требованиям конференции

## Примечание

В тексте есть пометка "(пустая строка)" - это означает, что нужно оставить пустую строку при оформлении в Word.
